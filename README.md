# Anime Showcase

A React e-commerce admin portal for anime products.

## Features
- Home landing page
- Product shop page
- Admin form to add products
- Search products
- Edit product price
- Delete products
- React Router navigation
- Custom hook for fetching products

## Setup
npm install

## Run App
npm run dev

## Run Backend
json-server --watch db.json --port 3001

## Run Tests
npm test
